# 메인(홈) 섹션 스펙 (SCR-01, SCR-02, 초과지출)

## 1. 요구사항 및 정책 (전문)

### SCR-01 (홈 빈 상태)

#### 📌 화면 목적
- 수입·지출 모두 없는 홈 화면.

#### 🔘 UI 요소
- 앱 타이틀 "AX 가계부" (좌상단)
- 월 네비게이션 ‹ N.DD ~ M.DD ›
- 빈 상태 일러스트
- "아직 입력한 내역이 없어요!" 타이틀
- "+버튼으로 지출 관리를 시작하세요" 서브텍스트
- FAB + 버튼 (우하단)
- BottomNavigation: 홈/통계/설정

#### 📐 정책
- 입력된 데이터 없음 => 이 화면 표시
- FAB + 탭 → SCR-03 이동
- 기본값: 현재 월 1일~말일 (매월 자동 갱신)
- ‹ › 탭 또는 스와이프로 이전/다음 주기 이동 (해당 주기도 내역 없으면 동일 빈 상태)
- 지출/수입 내역이 입력되면 SCR-02 화면 등장.

#### ➡️ 플로우
- FAB + 탭 → SCR-03 내역 추가

---

### SCR-02 (홈)

#### 📌 화면 목적
- 앱 진입 시 기본 화면. 현재 월(1일~말일) 기준 잔액·수입·지출 요약 및 거래 내역 표시

#### 🔘 UI 요소
- 앱 타이틀 (좌상단): "AX 가계부" 좌상단 고정
- 월 네비게이션 바: ‹ N.DD ~ M.DD › (기준일 1일이면 "N.1–N.말일", 그 외 기간 표기)
- 기간 텍스트: N.DD – M.DD (주기 시작일~종료일, 기준일 1일이면 N.1 – N.말일)
- 잔액 요약 카드: "수입 n" (해당 주기 수입 합산 총액, 고정 색상), "n원 사용했어요.", "n% 사용중", 예산 프로그레스 바
- 거래 내역 리스트: 날짜 헤더 + 현재 주기 전체 내역 무제한 스크롤. 지출 "-N" 검은색, 수입 "N" Primary 컬러
- FAB: 우하단 + 버튼
- BottomNavigation: 홈/통계/설정

#### 📐 정책
- HOME-004: 기본값 = 현재 월 1일~말일 (SET-002 기준일 미설정 시)
- SET-002 기준일 설정 시 해당 기준일 기반 주기로 전환
- ‹ › 탭 또는 좌우 스와이프로 이전/다음 주기 이동. 이동 범위 제한 없음
- 수입 n = 해당 주기 수입 내역 합산 총액. 색상 고정 (초과 여부 무관)
- HOME-003: 수입 대비 사용 80%↑ 주황, 100%↑ 빨강. 수입 0 + 지출 있음 → 즉시 100% 초과(빨강)
- 거래 내역 리스트 정렬: 최신 등록 순

#### 📊 상태별 표시 정책
- **수입·지출 모두 없음**
  → SCR-01 빈 상태

- **수입 또는 지출 중 하나라도 있음**
  → SCR-02 표시

- **수입 n = 수입 입력값 (없으면 0)**
  → 카드 상단 "수입 n" 표시 (수입 합산 총액, 색상 고정)

- **수입 0 + 지출 있음**
  → 바 100% 빨강 고정 + "N원 초과했어요." 텍스트

- **지출 ≤ 수입 (수입 > 0)**
  → 프로그레스 바 정상 (80%↑ 주황, 100% = 빨강)

- **지출 > 수입 (수입 > 0)**
  → 바 100% 빨강 고정 + "N원 초과했어요." 텍스트

#### ➡️ 플로우
- FAB + 탭 → SCR-03 내역 추가
- 거래 항목 탭 → SCR-04 상세내역
