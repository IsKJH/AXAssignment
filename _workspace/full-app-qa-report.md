## 정상 확인 목록 (간단히)

- 홈: 월 이동 ‹ › 연속 이동(6월→2월→6월) 후 데이터 정합성 유지, 2월 28일(평년) 정확, 빈 월 빈 상태("아직 입력한 내역이 없어요!") 정상, 거래 탭→상세 진입, FAB→추가 진입
- 요약 카드: 수입 0일 때 "100% 사용중"+빨간 바, 수입 2억/지출 1.002억일 때 "50% 사용중"+노란 바 — 비율 계산 정확
- 거래 추가: 지출/수입 토글(라벨·카테고리 기본값 전환: 식비↔급여, 지출처↔수입처), 금액 콤마 포맷(1,000단위, 1억 정상), 0원+메모 입력 시 확인 버튼 비활성(저장 차단), 금액/메모 없으면 확인 비활성, 카테고리 선택→자동 복귀 반영(추가 플로우), 날짜/시간 휠(오늘/오전·오후/시/분) 선택·완료·닫기 정상, 저장 후 홈 즉시 반영
- 거래 상세: 표시 정보 정확(금액/카테고리/일시/메모), 편집 모드 진입, 금액·메모 수정 후 저장 → 홈 반영, 원복 정상
- 삭제: 삭제 확인 다이얼로그(취소/삭제) 정상, 취소 시 유지, 삭제 시 홈·합계 즉시 갱신
- 정기 등록: 체크박스 토글, (?) 탭 시 "매달 N일에 자동 등록할 수 있어요." 말풍선 표시/닫기 정상 (실제 자동 등록 동작은 장기 검증 필요로 미검증)
- 카테고리: 추가(이름 입력, 빈 이름 시 저장 비활성), 수정(프리필, 변경 없으면 저장 비활성), 삭제 확인 다이얼로그, 기본 카테고리는 수정/삭제 UI 자체가 없음(보호됨 — 정상으로 판단)
- 사용 중 카테고리 삭제: 해당 거래가 "미분류"로 전환되어 보존됨 (거래 손실 없음)
- 설정: 시작일 변경(1일→5일) 시 홈·통계 주기가 6월5일~7월4일로 즉시 반영, 1일 복원 확인. 리뷰 남기기(스토어 선택 표시), 앱 추천(공유 시트+문구), 사용 방법(4단계 가이드), 앱 버전 1.00 표시
- 통계: 주기 ‹ › 이동, 도넛/최다 소비 카테고리/남은 금액 계산 정확, 6개월 내역 진입/뒤로, 평균 소비 금액 계산 정확(100,224,000/6=16,704,000), 막대 탭 말풍선
- 시스템: 통계 탭에서 BACK → 홈 탭 이동(정상), 각 하위 화면 BACK → 직전 화면 복귀, 탭 전환 시 통계 주기 상태 유지
